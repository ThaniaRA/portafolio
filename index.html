Mis proyectos recientes

Desarrollo de un sitio web

El proyecto que estoy trabajando actualmente es el desarrollo de un sitio web para la clinica de fisioterapia "Prophisyo Huejutla".Este sitio principalmente funcionara para agendar Citas desde el sitio sin nesesidad de ir directamente a la clinica. Dentro de este sitio se podran visualizar los distintos servicios que ofrece, asi como un blog donde se mencionan las distintas tipo de lesiones que puede tener un paciente.

Desarrollo de una aplicacion movil

Actualmente estoy trabajando en el desarrollo de una aplicacion movil, para la clinica de fisioterapia "Prophysio Huejutla", en la cual se podran agendar citas,asi como podras modificar o eliminarla en caso de no poder asistir.

Desarrollo de un videojuego

Cabe mencionar que apenas se esta en la fase de diseño aun no hay un videojuego terminado.sin embargo este videojuego principalmente constara de 4 niveles y sus personajes principales son dos, una fisioterapeuta y un paciente. Tambien se trabaja en la parte del diseño de los niveles.
